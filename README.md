## ts-express

### Install Dependencies

```bash
$ yarn install
$ npm install
```

### 开发环境

```bash
$ yarn run dev
$ open http://localhost:7788/
```

### 生产环境

```bash
$ yarn run tsc
$ yarn start
```
### 配置文件 (后面完善或者修改 .env)

```bash
{
    port:'7788',
    mongodb_host:'localhost:27017',
    document_name:'test'
}
```
